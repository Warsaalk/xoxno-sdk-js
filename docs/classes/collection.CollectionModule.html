Class CollectionModule

CollectionModule provides a set of methods to interact with NFT collections. It includes methods for getting collection profile information, floor price, collection attributes, and searching NFTs within a collection.

Example

const xoxno = new XOXNO({ apiURL: 'https://api.xoxno.com', apiKey: 'your-api-key' });
const collectionModule = xoxno.collection;
